# META SCALE
Anti-ban social media automation engine with real-time risk scoring.
## Tech Stack
- **Backend**: Node.js + Express + TypeScript
- **Frontend**: React + WebSocket
- **Queue**: BullMQ + Redis
- **Database**: PostgreSQL / SQLite
## Getting Started

```bash
npm install
npm run dev
```
